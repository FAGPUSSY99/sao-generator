# template

A template for scaffolding out an SAO template.

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From repo

```bash
sao egoist/template-template template-foo
```

### From npm

```bash
sao template template-foo
```
